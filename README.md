## 使用上の注意点
- この現在公開されているものはβ版です．存分にバグが含まれていると考えられます．
- ランダムに生成したハンドから条件合致したものを出力するので，珍しいハンド(HCPが30点以上など)は生成に時間がかかる，もしくは生成に失敗することがあります．
## 今後実装したいと考えているもの
- ダブルダミー解析
- 珍しいハンドも生成できるアルゴリズム
- ハンドのローカルファイル入出力
- スマートフォン用ページ
